## 🧠 Core Concepts Overview

### 1️⃣ Variables
Think of variables as **boxes that store data**:
```java
int age = 25;              // Box named "age" stores the number 25
String name = "Ahad";      // Box named "name" stores text "Ahad"
boolean isStudent = true;  // Box named "isStudent" stores true
```

### 2️⃣ Data Types
Different types of boxes for different data:
```java
int age = 25;              // Whole numbers
double height = 5.9;       // Decimal numbers
String name = "Java";      // Text
boolean active = true;     // True or False
char grade = 'A';          // Single character
```

### 3️⃣ Operators
Tools to work with data:
```java
int sum = 5 + 3;           // Addition: 8
int diff = 10 - 4;         // Subtraction: 6
boolean isGreater = 5 > 3; // Comparison: true
```

### 4️⃣ Control Flow
Making decisions:
```java
if (age >= 18) {
    System.out.println("You are an adult");
} else {
    System.out.println("You are a minor");
}
```
